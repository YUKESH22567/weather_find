# Weather Dashboard

A JavaScript weather dashboard that fetches real-time weather data from the OpenWeatherMap API.

## Features

- City search with autofocus
- Displays temperature, humidity, wind speed, and weather description
- Weather icons from OpenWeatherMap
- Error handling for invalid cities and network issues
- Loading indicator during API requests
- Responsive glassmorphism design

## How to Use

1. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Open `script.js` and replace `YOUR_API_KEY_HERE` with your actual API key
3. Open `index.html` in a browser
4. Type a city name and click **Search**

## Concepts Covered

- Fetch API
- Async/await
- JSON parsing
- Error handling (try/catch)
- Dynamic DOM manipulation
- CSS glassmorphism & animations
